## ✨ 功能特性

### 🎯 核心功能
1. **自动跟随系统** - 默认根据操作系统设置自动切换
2. **手动切换** - 支持用户手动选择浅色/深色/自动
3. **持久化** - 用户偏好保存到 localStorage
4. **平滑过渡** - 0.3s 渐变动画

- ✅ iOS 13+ (支持系统深色模式)
- ✅ Android 10+ (支持系统深色模式)
- ✅ 所有现代浏览器 (Chrome, Firefox, Safari, Edge)
- ✅ 微信内置浏览器

## 🎮 使用方法

### 自动模式 (默认)
博客会自动根据系统设置切换：
- 系统设置为浅色 → 博客显示浅色
- 系统设置为深色 → 博客显示深色

### 手动切换
点击右下角的 🌙/☀️ 按钮：
1. 第一次点击：切换到深色模式
2. 第二次点击：切换到自动模式
3. 第三次点击：切换到浅色模式
4. 循环往复...

### 用户偏好
- 用户的主题选择会保存到 localStorage
- 下次访问时会记住用户的选择
- 清除浏览器缓存会重置为默认

## 🔧 技术实现

### CSS 变量
使用 CSS 变量实现主题切换，无需修改原有样式：
```css
[data-theme="dark"] {
  --white: #1a1a1b;
  --text-color: #E8E8E8;
  /* ...其他变量 */
}
```

### 媒体查询
自动模式使用 CSS 媒体查询检测系统设置：
```css
@media (prefers-color-scheme: dark) {
  [data-theme="auto"] {
    /* 深色模式变量 */
  }
}
```

### JavaScript
- 检测 localStorage 中的用户偏好
- 创建悬浮切换按钮
- 监听系统主题变化
- 添加动画反馈

## 🎨 自定义主题

如果要修改深色模式的颜色，编辑 `assets/css/dark-mode.css`：

```css
[data-theme="dark"] {
  --white: #1a1a1b;      /* 背景色 */
  --text-color: #E8E8E8; /* 文字颜色 */
  --link-color: #ff4d6a; /* 链接颜色 */
  /* ...修改其他颜色 */
}
```
